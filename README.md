# Welcome to Mini Message Board 📌

> A simple message board created with Express and MongoDB.

### 🌐 [Live Demo](https://message-board-mcstanley.fly.dev)

## Install

```sh
npm install
```

## Usage

### Environment variables (`.env`)

```
DATABASE_URI="<your_mongodb_connection_string>"
```

### Run command

```sh
npm run start
```
